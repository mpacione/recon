# Web UI Figma Intake

Drop Figma exports for the Recon WebUI here.

Useful files:

- Sticker sheet export as `png`, `svg`, or `pdf`
- Component/state exports for buttons, cards, tabs, tables, modals, forms
- Screenshots of target page layouts
- Design token notes for color, typography, spacing, borders, and radius

Prefer `2x` PNG exports for visual review and SVG exports when exact sizing or vector structure matters.
